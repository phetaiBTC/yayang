# Bookstore Management System

OpenSpec project. Stack: NestJS + MikroORM (back/) + Vue 3 + PrimeVue (front/).
Covers master data, registration (OTP), purchasing, importing, selling,
reservation, and reporting.

## Capabilities
master-data, registration, purchasing, importing, selling, reservation, reporting
